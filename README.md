# Mini Social Platform

A modern social media web application for managing posts and comments. The project is built with React on the frontend and Node.js on the backend.

## Key Features
- **Post Management:** Create, delete, and view posts.
- **Comments System:** Expandable/collapsible comment sections for each post.
- **Pagination:** Efficient loading of posts (5 per page) and comments (3 per page) using a "Load More" functionality.
- **Search & Filter:** Search posts by title or ID, and filter posts by user ownership.
- **Modern UI:** Clean, responsive design with a card-based interface.

## Tech Stack
- **Frontend:** React, React Router, Axios.
- **Backend:** Node.js, Express, MySQL.
- **Styling:** CSS3 with custom component-based styling.
